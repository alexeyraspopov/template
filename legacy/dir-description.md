# AngularJS Directives

## ngController

 - create new scope
 - extend model

## ngHide / ngShow

 - subscribe value change
 - update element by value

## ngIf

 - create new scope
 - subscribe value change
 - update element by value

## ngInclude

 - load partial by value
 - create new scope
 - update element by loaded value

## ngRepeat

 - subscribe value change
 - create new scope
 - update element

## ngStyle

 - subscribe value change
 - update element by value
